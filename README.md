## Deployment

Install `npm`

```shell
npm install
```

### Dev

```shell
npm run dev
```

1. Watch SCSS files
2. Then compile to CSS for preview

### Prod

```shell
npm run prod
```

1. Compile to CSS
2. Merge CSS
3. Minify CSS
